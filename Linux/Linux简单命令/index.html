¶常用Linux命令的基本使用

序号 命令 对应英文 作用
01 ls list 查看当前文件夹下的内容
02 pwd print work directory 查看当前所在文件夹
03 cd[目录名] change directory 切换文件夹
04 touch[文件名] touch 如果文件不存在，新建文件
05 mkdir[目录名] make directory 创建目录
06 rm[文件名] remove 删除指定的文件名
07 clear clear 清屏

小技巧

  • ctrl + shift + =放大终端窗口的字体显示
  • ctrl + -缩小终端窗口的字体显示

¶Linux终端命令格式

¶终端命令格式

解释:

  • command :命令名，相应功能的英文单词火单词的缩写
  • [-options]:选项，可用来对命令进行控制，也可以省略
  • parameter：传给命令的参数，可以是零个，一个或者多个

[]代表可选

¶查阅帮助信息

¶–help

说明：

  • 提示command命令中的帮助信息

¶man

man command

说明：

  • 查阅command命令的使用手册

man是manual的缩写，是Linux的一个手册，包含了绝大部分的命令，函数的详细说明

使用man时的操作间

操作键盘 功能
空格键 显示手册的下一屏
Enter键 一次滚动手册页的一行
b 回滚一屏
f 前滚一屏
q 退出

¶1.1终端使用技巧

¶自动补全

  • 在敲出文件/目录/命令的前几个字母之后，按下tab键盘
    • 如果输入的没有歧义，系统会自动补全
    • 如果还存在其他文件/目录/命令，在按一下tab键，系统会提示可能存在的命令

¶曾经使用过的命令

  • 按上/下光标可以在曾经使用过的命令之间来回切换
  • 如果想要退出选择，并且不想执行当前选中的命令，可以按ctrl + c
